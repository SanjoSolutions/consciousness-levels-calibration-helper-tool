Consciousness Levels Calibration Helper Tool

This work is devoted to God.

Reference scale: The Map of the Scale of Consciousness by David R. Hawkins (see on the right side or bottom on mobile or one of the books by David R. Hawkins)

Also see: How to calibrate the levels of Consciousness

About this tool:
This tool can help to encode a consciousness level as a number in the decimal system
which has been calibrated with a consciousness calibration technique
by verifying the digits of the number encoded as a binary number
from the least significant bit to the most significant bit.

Tool

Recommended steps:

1. By declaration: this calibration is devoted to God.
2. In the name of the highest good, I have permission to calibrate the consciousness level of <thing that is intended to be calibrated>.
3. In the name of the highest good, I have permission to ask, if there are more digits to calibrate.
4. If "3." calibrated as true: In the name of the highest good, there are more digits to calibrate.
5. If "4." calibrated as true: In the name of the highest good, I have permission to ask, if the next digit is a 1.
6. If "5." calibrated as true: In the name of the highest good, the next digit is a 1.
7. If "5." or "6." calibrated as untrue: In the name of the highest good, I have permission to ask, if the next digit is a 0.
8. If "7." calibrated as true: In the name of the highest good, the next digit is a 0.
